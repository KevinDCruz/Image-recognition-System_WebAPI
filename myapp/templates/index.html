About the Project

This Project focusses on the Computer Vision domain of Computer Science and uses Deep Learning techniques for Image Recognition of common objects. Convolutional Neural Networks plays a major role in Deep Learning and forms the base for this system.
The Backend Technology used in this project consists of Keras, an API designed for Neural Networks and TensorFlow, Google's framework for Machine Learning. The Models trained include ResNet50 and VGG19, where the weights are from Imagenet. The WEB API is built using Django and Bootstrap. It shows a bar chart of the top 5 predictions of the Input Image along with it's probabilities.

Technology Used:
Backend: Python3, Libraries: (Keras, TensorFlow, NumPy, Pandas, MatplotLib)
Frontend: Django, Bootstrap (HTML, CSS, JavaScript)
