About the Project

This Project focusses on the Computer Vision domain of Computer Science and uses Deep Learning techniques for Image Recognition. Convolutional Neural Networks plays a major role in Deep Learning and forms the base for this system.
The system uses Keras, which is a Neural Network API built in Python with the backend of TensorFlow. It shows the top 5 predictions of the Input Image and returns a bar graph showing the probabilities of the same. The Image can be uploaded from a local disk or by copying it's URL.

Technology Used:
Backend: Python (Keras, TensorFlow, NumPy, Pandas, MatplotLib)
User Interface: HTML, CSS, JavaScript, Bootstrap, Django
